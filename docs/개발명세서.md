# ⚽ KickPot (킥팟) MVP 기능 명세

---

## 1. Auth (인증)

- [ ] 회원가입
- [ ] 로그인
- [ ] 로그아웃
- [ ] 프로필 생성 (닉네임, 포지션 등)

---

## 2. User (개인 프로필)

- [ ] 프로필 조회
- [ ] 프로필 수정
  - [ ] 프로필 사진 업로드
  - [ ] 자기소개 입력
  - [ ] 주 포지션 설정
  - [ ] 선호 포지션 설정
- [ ] 유저 경기 기록 조회 (연동)

---

## 3. Team (팀)

### 기본
- [ ] 팀 생성
- [ ] 팀 삭제
- [ ] 팀 가입 (초대 링크/코드)
- [ ] 팀 탈퇴
- [ ] 팀원 강퇴

### 팀 설정
- [ ] 팀 이름 수정
- [ ] 팀 로고 업로드
- [ ] 팀 배경 이미지 업로드
- [ ] 팀 소개 수정

### 팀 조회
- [ ] 팀 목록 조회
- [ ] 팀 상세 조회

---

## 4. Team Feed (팀 게시판)

- [ ] 게시글 생성
- [ ] 게시글 수정
- [ ] 게시글 삭제
- [ ] 게시글 목록 조회
- [ ] 댓글 생성
- [ ] 댓글 삭제
- [ ] 공지 설정

---

## 5. Vote (투표)

- [ ] 투표 생성
  - [ ] 날짜 투표
  - [ ] 참여자 투표
- [ ] 투표 참여
- [ ] 투표 결과 조회

---

## 6. Match (매칭)

### 매칭 생성
- [ ] 매칭 생성 (날짜, 장소, 팀)

### 매칭 요청
- [ ] 상대 팀에 매칭 요청
- [ ] 매칭 요청 수락
- [ ] 매칭 요청 거절

### 매칭 상태
- [ ] 요청됨
- [ ] 대기중
- [ ] 확정

---

## 7. Participant (참여자 관리)

- [ ] 참여자 투표 생성
- [ ] 참여자 확정
- [ ] 참여자 목록 조회

---

## 8. Lineup (라인업 / 포지션 배치)

- [ ] 라인업 생성
- [ ] 참여자 리스트 불러오기
- [ ] 포지션 배치 (GK, DF, MF, FW)
- [ ] 라인업 수정
- [ ] 라인업 조회

---

## 9. Record (경기 기록)

### 기록 입력
- [ ] 경기 결과 입력 (스코어)
- [ ] 득점자 등록
- [ ] 어시스트 등록

### 기록 승인
- [ ] 상대팀 승인 요청
- [ ] 승인
- [ ] 거절 (수정 요청)

### 상태 관리
- [ ] 작성됨
- [ ] 검토중
- [ ] 수정요청
- [ ] 확정

---

## 10. Notification (알림)

- [ ] 팀 초대 알림
- [ ] 매칭 요청 알림
- [ ] 투표 알림
- [ ] 기록 승인 요청 알림
- [ ] 댓글 알림

---

## 🔥 핵심 플로우 체크

- [ ] 팀 생성 → 팀원 초대
- [ ] 투표 → 경기 날짜 확정
- [ ] 매칭 요청 → 상대팀 수락
- [ ] 참여자 투표 → 참여자 확정
- [ ] 라인업 구성
- [ ] 경기 기록 입력 → 상대 승인 → 확정

---

## 🚀 MVP 우선순위

### P0 (필수)
- Auth
- Team
- Match
- Vote
- Record

### P1 (중요)
- Lineup
- Notification

### P2 (추후)
- 상세 통계
- 랭킹 시스템
